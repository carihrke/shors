# Classical and Quantum Approaches to Factorization

**Authors:** Carson Ihrke, Will Geiger  

---

## Overview
This repository contains the source code and experimental framework developed to analyze the mathematical security boundaries of the RSA encryption scheme. The project establishes performance baselines using purely classical brute-force prime factorization methodologies and contrasts them against a generalized quantum circuit simulation executing Shor's Algorithm. 

By charting execution times across scaling key sizes, this study maps the "computational wall" that classical systems encounter and demonstrates how quantum mechanics (superposition, entanglement, and wave interference) can inherently bypass these cryptographic constraints.
